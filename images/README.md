# Images Directory

## Profile Photo

Please add your profile photo here with the filename: `rakesh-photo.jpg`

### Instructions:

1. **Save your photo** as `rakesh-photo.jpg` in this `images/` directory
2. **Recommended specifications:**
   - Format: JPG or PNG
   - Size: At least 800x800 pixels (square format works best)
   - File size: Under 500KB for fast loading
   - Aspect ratio: 1:1 (square) for best circular display

### Image Requirements:

- **Format:** JPG, PNG, or WebP
- **Dimensions:** Minimum 400x400px, recommended 800x800px or larger
- **File size:** Keep under 500KB for optimal performance
- **Content:** Professional headshot or portrait photo

### Alternative Formats:

If you use a different filename, update the `src` attribute in `index.html`:
```html
<img src="images/your-filename.jpg" alt="Rakesh Babu Gogineni" class="profile-photo">
```

### Fallback:

If the image is not found, the portfolio will automatically show the "RBG" placeholder with gradient background.
